# Image Generation Prompts for M365 Copilot / Designer

Use these prompts in Microsoft Designer (https://designer.microsoft.com) or M365 Copilot with image generation to create marketing visuals.

---

## Hero Banner (16:9, landscape)

**Prompt:**
```
A dramatic colonial-era trading ship (similar to a galleon) with golden sails, sailing through a digital ocean of data streams and floating enterprise icons (CRM dashboard, IT ticket, purchase order, email metrics). The ship's sails display logos representing Salesforce (blue cloud), ServiceNow (green circle), SAP (blue hexagon), and HubSpot (orange sprocket). The sky transitions from a warm golden sunset on the left to a futuristic blue digital grid on the right, symbolizing the bridge between traditional enterprise and AI. Style: cinematic digital illustration, rich colors, dramatic lighting. Text overlay area at the bottom. No text in the image.
```

## Widget Showcase (4 panels, 2x2 grid)

**Prompt:**
```
Four panels in a 2x2 grid, each showing a different enterprise widget design:
Top-left: A Salesforce-styled data table with blue header, pill-shaped status badges (blue, amber, green), and an edit pencil icon. Lightning Design System look.
Top-right: A ServiceNow-styled incident list with teal header, priority indicators (red P1, orange P2, gold P3, green P4), and expandable rows.
Bottom-left: An SAP-styled purchase order view with dark navy shell bar, Fiori semantic badges, and a materials detail card.
Bottom-right: A HubSpot-styled email dashboard with coral accents, metric cards showing open rates and click rates with percentage bars, and a contact list drill-down.
Style: UI mockup, realistic Fluent components, clean shadows, dark and light theme variants. No text labels.
```

## Tips for Best Results

1. **Microsoft Designer** (designer.microsoft.com) gives the best results for marketing visuals
2. **M365 Copilot Image Creator** works for quick social media images
3. Add "professional, enterprise, SaaS marketing style" to any prompt for polished results
4. Add "no text, no watermark" if you want to overlay your own text later
5. Generate at highest resolution available, then crop to needed aspect ratio
