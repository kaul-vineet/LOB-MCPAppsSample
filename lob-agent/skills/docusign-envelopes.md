# DocuSign eSignature
**LOB:** DocuSign  
**Starter:** "Show me my recent DocuSign envelopes. I need to check which ones are still awaiting signatures."

## What this skill does
- Lists envelopes with status badges (sent, delivered, completed, voided)
- Drills into envelope detail — shows recipients and signing progress
- Lists available templates for sending new envelopes
- Sends new envelopes from a template via interactive form
- Voids, resends, and generates embedded signing URLs

## Example prompts
- "Show me my DocuSign envelopes"
- "Show me envelopes awaiting signature"
- "Show me the details of envelope env-0002-gtc"
- "List my signing templates"
- "Send the NDA template to james@gtc.internal for signature"
- "Void envelope env-0004-gtc — the terms have changed"
