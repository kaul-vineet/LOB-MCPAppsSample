# Flight Tracker
**LOB:** Flight Tracker (OpenSky Network)  
**Starter:** "Show me departures from Heathrow (EGLL) today. I also want to check the live state of a specific aircraft."

## What this skill does
- Shows live departures and arrivals at any ICAO airport (e.g. EGLL, KJFK, OMDB)
- Looks up the live state (position, altitude, speed) of any aircraft by ICAO24 code
- Retrieves flight history for a specific aircraft over a date range
- Shows flight track waypoints for a given flight

## Example prompts
- "Show me departures from Heathrow today"
- "Show me arrivals at JFK yesterday"
- "What is the live state of aircraft 4ca2bf?"
- "Show me the flight history for 4ca2bf over the last two days"
- "Show me the track for aircraft 3c675a"
