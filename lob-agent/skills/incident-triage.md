# IT Incident Triage
**LOB:** ServiceNow  
**Starter:** "Show me open incidents sorted by priority. I need to check the critical ones and update their status."

## What this skill does
- Lists open incidents with P1–P4 priority colour coding
- Supports inline status updates and work note additions
- Drills into service requests and their fulfilment items
- Allows creating new incidents directly from the widget

## Example prompts
- "Show me open incidents"
- "Raise a new incident — VPN not connecting for Building A users"
- "Show me service requests pending approval"
- "Add a work note to INC0010042 saying the network team is investigating"
- "Resolve incident INC0010038 — issue fixed by patch"
